# Reference Material

This directory contains frozen research inputs used to design the new SaaS booking platform.

- `amelia-source/`: Amelia Premium WordPress plugin source used as behavioral and architectural inspiration.
- `graphify/saas-core/`: focused Graphify output recommended for SaaS architecture reference.
- `graphify/backend/`: backend-focused Graphify output for deeper PHP/domain/persistence investigation.

Do not treat this directory as source code for the new SaaS. New implementation should be driven by `specs/`, `.specify/memory/constitution.md`, and the current Spec Kit plan.
